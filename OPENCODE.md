# OpenCode: навигация и горячие клавиши

## Режимы
| Действие | Клавиша |
|----------|---------|
| Plan-режим (без правок) | `Tab` |
| Build-режим (с правками) | `Tab` (повторно) |

## Команды (ввод через `/`)
| Команда | Что делает |
|---------|------------|
| `/init` | Анализ проекта → AGENTS.md |
| `/connect` | Подключить LLM-провайдера |
| `/undo` | Отменить последнее изменение |
| `/redo` | Вернуть отменённое изменение |
| `/share` | Поделиться сессией (ссылкой) |
| `/help` | Справка по opencode |

## Навигация по проекту
| Действие | Как |
|----------|-----|
| Fuzzy-поиск файла | `@` + начало имени |
| Прикрепить изображение | Перетащить в терминал |

## Диагностика
| Команда | Что делает |
|---------|------------|
| `opencode debug config` | Показать итоговый конфиг |
| `opencode --log-level DEBUG` | Запуск с детальными логами |
| `opencode --print-logs` | Вывод логов в терминал |
| `opencode models` | Список доступных моделей |
| `opencode upgrade` | Обновить opencode |

## Где что лежит
| Что | Путь |
|-----|------|
| Глобальный конфиг | `~/.config/opencode/opencode.json` |
| Проектный конфиг | `opencode.json` (корень проекта) |
| Логи (Windows) | `%USERPROFILE%\.local\share\opencode\log\` |
| Данные сессий | `%USERPROFILE%\.local\share\opencode\project\` |

## Permissions (напоминание)
- `*` = `ask` (всё требует подтверждения)
- `read`, `glob`, `grep`, `lsp`, `websearch` = `allow` (без спроса)
- Остальное (`bash`, `edit`, `task`, `webfetch`...) = спрашивает
